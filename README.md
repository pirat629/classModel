Телеграмм бот для определения заболевания растения по фото и описанию листа. 

В работе представлена гибридная модель классификации, где в качестве модели для работы с текстом используется модель EfficientNet B0, а для работы с изобрежениями используется модель Birt. 

В работе предствлены графики, покзывающие ошибки обучения на валидационных и тестовых данных.
